# Captures d'ecran

Ce dossier est prevu pour les captures de la demonstration Streamlit.

Captures conseillees :

- `streamlit_accueil.png` : interface au lancement ;
- `question_carte_perdue.png` : question saisie dans le chat ;
- `resultat_intention.png` : categorie detectee, score et exemple proche ;
- `categories.png` : sidebar avec le filtre de categories.

Commande pour lancer l'application :

```bash
streamlit run app.py
```

Puis ouvrir :

```text
http://localhost:8501
```
